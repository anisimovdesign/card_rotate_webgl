# Настройка голографического эффекта

Все параметры находятся в `index.html`, в шейдере `holoFS` (секция `void main()`).

---

## Маска (порог глянца)

```glsl
maskVal = smoothstep(0.1, 0.35, maskVal);
```

- `0.1` — нижний порог. Чем выше, тем жёстче обрезаются полупрозрачные края маски (убирает белую обводку). Диапазон: `0.03`–`0.15`
- `0.35` — верхний порог. Чем ниже, тем резче граница глянца/мата

---

## Возмущение нормалей (шершавость глянца)

```glsl
float px = grain(vUV * 50.0);
float py = grain(vUV * 50.0 + vec2(73.1, 19.3));
vec3 perturbedN = normalize(N + vec3((px - 0.5) * 0.16, (py - 0.5) * 0.16, 0.0) * maskVal);
```

- `50.0` — размер зерна возмущения. Больше = мельче микрорельеф, меньше = крупнее
- `0.16` — амплитуда возмущения. Больше = сильнее рассеивание света, меньше = ближе к зеркалу. Диапазон: `0.02`–`0.2`

---

## Спекуляр (блик)

```glsl
float spec = pow(max(dot(perturbedN, H), 0.0), 14.0) * 0.12;
```

- `14.0` — резкость блика. Больше = уже и резче (зеркало), меньше = шире и мягче (полуглянец). Диапазон: `8`–`64`
- `0.12` — интенсивность блика. Диапазон: `0.05`–`0.3`

---

## Радуга (цвет голографии)

### Mesh-паттерн (форма пятен)

```glsl
float n1 = sin(vUV.x * 3.2 + vUV.y * 1.8 + a * 1.4);
float n2 = sin(vUV.x * 1.4 - vUV.y * 3.5 + a * 0.9 + 1.7);
float n3 = cos(vUV.x * 2.6 + vUV.y * 0.7 - a * 1.1 + 0.5);
float n4 = sin(vUV.x * 0.8 + vUV.y * 2.4 + a * 0.6 + 3.1);
```

4 волны создают органичный mesh-градиент. Первые два числа в каждой строке — частоты по X и Y (форма пятен), последнее число с `a *` — скорость сдвига при вращении.

### Hue (оттенок)

```glsl
float hue = fract((n1 + n2 + n3 + n4) * 0.12 + a * 0.15);
```

- `0.12` — контрастность цветовых пятен. Больше = резче переходы цветов
- `0.15` — скорость смены цветов при вращении

### Насыщенность

```glsl
float sat = (0.42 + 0.12 * (...)) * hlFade;
```

- `0.42` — базовая насыщенность радуги на глянце. Диапазон: `0.2`–`0.6`
- `0.12` — вариация насыщенности

### Яркость

```glsl
float val = 0.95 + 0.05 * n3;
```

- `0.95` — базовая яркость. Диапазон: `0.8`–`1.0`
- `0.05` — вариация

---

## Highlight (скользящее пятно радуги)

```glsl
float hlX = 0.5 + sin(a + 0.1) * 0.45;
float hlY = 0.5 + cos(a * 0.7 + 3.2) * 0.3;
float hlDist = length(vUV - vec2(hlX, hlY));
float hlFade = 1.0 - smoothstep(0.15, 0.55, hlDist);
```

- `0.1` / `3.2` — стартовая фаза по X / Y (начальная позиция пятна при загрузке)
- `0.45` / `0.3` — амплитуда движения по X / Y. Больше = шире размах
- `0.15, 0.55` — размер пятна. `0.15` = ядро (полная радуга), `0.55` = край (затухание). Увеличить оба = больше пятно

### Подмешивание

```glsl
float glossStrength = (0.5 + fresnel * 0.55) * mix(0.3, 1.0, hlFade);
```

- `0.5` — базовая сила подмешивания радуги к текстуре (50%). Диапазон: `0.3`–`0.7`
- `0.55` — добавка на краях (fresnel)
- `0.3` — минимум подмешивания вне highlight. `0.0` = полностью серебристый вне пятна

---

## Блик-вспышка (flare)

```glsl
float wX = vUV.x + sin(vUV.y * 5.0 + a * 1.3) * 0.12;
float wY = vUV.y + cos(vUV.x * 4.0 + a * 0.9) * 0.1;
float flare = pow(max(1.0 - abs(fract(wX * 0.8 - wY * 0.3 + a * 0.5) - 0.5) * 1.0, 0.0), 1.5) * 0.1;
```

- `0.12` / `0.1` — сила искажения формы (органичность). `0` = прямая полоса
- `1.0` (после `* `) — ширина блика. **Меньше = шире**, больше = уже. Диапазон: `0.8`–`4.0`
- `1.5` — резкость спада. Меньше = мягче и шире, больше = резче
- `0.1` — интенсивность (яркость). Диапазон: `0.05`–`0.5`
- `0.5` (в `a * 0.5`) — скорость движения при вращении

---

## Зернистость

### Глянец (маска)

```glsl
float glossGrain = (grain(vUV * 600.0 + ...) - 0.5) * 0.025;
```

- `600.0` — размер зерна. Больше = мельче
- `0.025` — интенсивность (+-1.25%). Диапазон: `0.005`–`0.04`

### Матовая часть (фон карты)

```glsl
float g = (grain(vUV * 800.0) - 0.5) * 0.03;
```

- `800.0` — размер зерна
- `0.03` — интенсивность (+-1.5%). Диапазон: `0.01`–`0.05`

---

## Матовая голография (фон карты вне маски)

```glsl
vec3 matteRainbow = hsv2rgb(vec3(hue, sat * 0.1, val * 0.95));
vec3 matteHolo = mix(baseColor, matteRainbow, 0.05) + vec3(g);
```

- `sat * 0.1` — насыщенность = 10% от глянца
- `0.05` — подмешивание (5%). Диапазон: `0.0`–`0.15`
